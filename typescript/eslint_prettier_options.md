prettierのオプション一覧

# 表記について
見出しは設定ファイル上の表記にしています。セミコロンの後は設定値のデータ型。

CLI表記とは、CLIで指定する場合のオプション名。

# オプション一覧

### printWidth: int
- CLI表記: `--print-width`
- デフォルト: 80
- 行の折り返しの長さ。

### tabWidth: int
- CLI表記: `--tab-width`
- デフォルト: 2
- インデントのスペース数。

### useTabs: bool
- CLI表記: `--use-tabs`
- デフォルト: false
- スペースではなくタブでインデントする。

### semi: `bool`
- CLI表記: `--no-semi`
- デフォルト: true
- ステートメントの最後にセミコロンを付与。

### singleQuote: `bool`
- CLI表記: `--single-quote`
- デフォルト: false
- 二重引用符の代わりに単一引用符を使用する。
