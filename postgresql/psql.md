psqlの使い方（ライトユーザー向け）


# 🚀起動
```
$ psql -h {接続先アドレス} -p {接続先ポート} -U {接続ユーザー}
```

### 例：ローカル接続
```
$ psql -h localhost -p 5432 -U postgres
```
※`-h`は省略可。
※`-p`も、デフォルト（5432）であれば省略可。

# psqlの終了
```
# \q
```

# 👀データベース一覧の確認
```
# \l
```

# 🛠️データベース作成

```
# create database {データベース名}
```

### 例：`sample`という名前のデータベースを作成

```
# create database sample;
```

# カレントデータベースの変更

```
# \c {データベース名}
```

### 例：`sample`データベースへ変更

```
# \c sample
```

# 👀テーブル一覧の確認
```
# \z
```

# 👀テーブル定義の確認
```
# \d {テーブル名}
```
