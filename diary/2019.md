# 2019-03-06 Wed
### サーバー証明書
- PKI
  - Public Key Infrastructureの略。公開鍵基盤。認証局、証明書、リポジトリで構成される。このPKIの規格がX.509。
- 認証局（CA）
  - 証明書を発行できる、信頼できる機関
- 証明書（公開鍵証明書）
  - 認証局から発行される。発行者（issuer:発行者CA）、有効期限（validity）、サブジェクト（subject:発行先）、公開鍵情報、署名アルゴリズム(Signature Algorithm)、署名(Signature)等から構成される。
- リポジトリ
  - 認証局が業務を運営するにあたっての公開情報が格納されている場所。証明書や証明書失効リスト（CRL:certificate revocation list）等がある

### TSA証明書チェーンの作成
- ①秘密鍵tsaroot.keyを作成
  - openssl genrsa -out tsaroot.key 2048 -config openssl.cnf
- ②秘密鍵tsaroot.keyを元に、CSR（公開鍵やコモンネーム等）作成、自己署名を同時に行い、サーバー証明書(tsaroot.crt)を作成
  - openssl req -new -x509 -key tsaroot.key -out tsaroot.crt -config config/tsaroot.cnf
- ③秘密鍵tsa.keyを作成
  - openssl genrsa -out tsa.key 2048 -config openssl.cnf
