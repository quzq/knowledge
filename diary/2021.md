# 2021-05-13 Thr
- elasticsearch
  - analyzer
    - Character filters: 文字列に対し、Tokenizerで分割する前に必要な処理（追加、削除、変更）を行う。前処理的ポジションで、使用は任意。
    - tokenizer:文字列を単語レベルに分割する役割を持つ。必須項目。
    - token filers:Tokenizerで分割された内容に対し、必要な処理（追加、削除、変更）を行う。後処理的ポジションで、使用は任意。
